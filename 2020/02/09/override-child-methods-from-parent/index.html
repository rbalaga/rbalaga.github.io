Overriding Child Component Functions From Parent Component.

Feb 09, 2020

When i was trying to customize functionality of some components from grommet library without disturbing their implementation, I was thought of creating a wrapping component to library component and changing the library component functions from wrapped component will solve my problem.

above component will display temperature in fahrenheit. when we do not have acess to modify the code of the above component, but we required to display the temperature in Celsius using the Same component, then we can override displayTemp method from its parent component and we can introduce our functionality.

we can do so by registering a ref to Temperature component and then we can modify prototype of the component by overriding properties/functions of Temperature component.

in above code we had used Temperature component but we had overritten displayTemp method of Temperature component to display it in Celsius.
